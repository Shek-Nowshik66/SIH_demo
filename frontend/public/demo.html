CivicResolve Demo

Smart India Hackathon 2025 - Citizen Reporting Platform

How to Demo This Application

This is a complete HTML/CSS/JavaScript application for SIH demo. Follow these steps:

  1. Start with Landing Page: Shows platform overview and technology stack
  2. Report an Issue: Fill out the citizen complaint form with photo and location
  3. View Dashboard: See all complaints as a government official with filtering options
  4. Demonstrate Features: Show real-time updates using localStorage persistence

Technology Stack Implemented

This demo showcases integration of modern technologies:

Frontend
HTML5, CSS3, JavaScript ES6+
UI/UX
Modern responsive design with animations
Data Storage
Browser localStorage for persistence
Location Services
GPS geolocation API
Charts & Analytics
Chart.js for dashboard visualizations
Icons
Lucide React icon library

Key Features Demonstrated

  • Citizen Reporting: Photo upload, GPS location, issue classification
  • Government Dashboard: Real-time complaint tracking and filtering
  • AI Classification: Automatic department assignment based on issue type
  • Priority Management: High, Medium, Low priority handling
  • Status Tracking: Pending → In Progress → Resolved workflow
  • Analytics: Charts showing complaint distribution and trends
  • Responsive Design: Works on mobile and desktop devices

Demo Flow for SIH Presentation

  1. Landing Page: Explain the problem and solution overview
  2. Technology Stack: Highlight modern tech integration
  3. Citizen Flow: Report a sample complaint with photo and location
  4. Government Flow: Show dashboard with sample + new complaint
  5. Filtering & Management: Demonstrate status updates and filtering
  6. Analytics: Show charts and metrics for decision making
